# JP's Promox Scripts

This is a collection of scripts that I'm using to quickly set up Proxmox
hypervisors for demos and live Kubernetes classes.

`stage-1-install-proxmox` = scripts to automated installation of Proxmox
on Hetzner dedicated servers.

`stage-2-configure-proxmox` = not much right now; will dump more scripts
there later.

`stage-3-use-proxmox` = scripts and Terraform configurations that I
used for a few demos and livestreams, including [that one][yt] (in French).
